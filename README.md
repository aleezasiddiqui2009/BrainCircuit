 BrainCircuit

BrainCircuit is an AIpowered math puzzle game where reinforcement learning agents  
(Q learning & MonteCarlo Tree Search) learn optimal strategies over time.

Features
- *AI Agents:* Q Learning agent + MCTS agent  
- *Puzzle Environment:* Custom math-based puzzle environment  
- *Training Dashboard:* Visual performance graphs  
- *Human vs AI Mode:* Play against trained agents  
- *Modular Code:* Easy to upgrade and experiment with RL algorithms
